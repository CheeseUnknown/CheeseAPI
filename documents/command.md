# **命令**

```bash
$ CheeseAPI -h
usage: CheeseAPI [-h] [--app [APP]] [--host [HOST]] [--port [PORT]] [--reload [RELOAD]] [--workers [WORKERS]]

options:
  -h, --help           show this help message and exit
  --app [APP]          【默认值：app:app】
  --host [HOST]        【默认值：127.0.0.1】
  --port [PORT]        【默认值：5214】
  --reload [RELOAD]    【默认值：False】
  --workers [WORKERS]  workers为0时会自动设置为cpu核数*2【默认值：1】
```
